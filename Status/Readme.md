# Status

## Usage(s)

Set the status for a given task. In this case, status was used to determine where in the work process the task was.

- Backlog
- Ready for Dev
- In progress
- Completed
- Blocked

## Icons

- (Backlog)
- (Ready for Dev)
- (In progress)
- (Completed)
- (Blocked)

## Interactions

- Inline edit using dropdown menu
